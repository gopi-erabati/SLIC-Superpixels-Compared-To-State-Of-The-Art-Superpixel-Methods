Manual for GS04 algorithm

1. Change your current working directory of MATLAB to this folder(GS04).
2. Run "superpixel_gs04(in_img,sigma,k,min_size)
	The description of parameters is given in superpixel_gs04.m file
	you can read it by running "help superpixel_gs04"

	An example is given at end of help document
3. You can see superpixels in a new figure window

Cheers!
