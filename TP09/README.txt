
Manual for superpixel TP09 (on a 64-bit machine)

1. Change the current working directory of MATLAB to this folder.
2. Run "superpixel_tp09(in_img,num)" 
	The description of parameters is given in superpixel_tp09.m
	You can read it by running "help superpixel_tp09"

	An example is given at end of help file

	This algorithm takes more time depending on your system. Have patience!
3. You can see the superpixels in a figure window

Cheers!
