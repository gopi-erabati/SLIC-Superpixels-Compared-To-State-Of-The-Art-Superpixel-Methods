Manual for QS09 algorithm

1. change your current working directory of MATLAB to this folder.
2. Run "run('./toolbox/vl_setup')" to setup vlfeat library.
3. Run "superpixel_qs09(in_img,ratio,kernel,maxdist,ndist)
	The description of parameters is given in superpixel_qs09.m file
	You can read by running "help superpixel_qs09"

	An example is given at end of help file

4. After you run you wil get superpixels with image.

Cheers!
